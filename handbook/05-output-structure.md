# 输出结构(Output Structure)

**版本:** 0.1.0
**状态:** draft

---

## 1. 本章的用途

本章描述**当前引擎实际生成的客户剧本结构**。

这里的权威依据是:

- `engine.markdown_renderer`
- 当前 CLI / Web 生成结果
- 对应单元测试

若历史示例或旧版方法论文档与本章冲突,以本章和当前渲染行为为准。

---

## 2. 当前剧本是什么

当前剧本不是传统资产配置报告,也不是纯 narrative 的五段散文。它是一份**顾问交付给客户的规划剧本**,同时承担 3 个任务:

- 解释当前家庭状态
- 展示未来节点压力
- 给出可执行的资金分层与回顾边界

因此,当前剧本天然比“high-level 骨架”更具体,但仍然**不构成投资建议**。

---

## 3. 元数据

当前剧本开头稳定包含以下信息:

| 字段 | 说明 |
|---|---|
| 标题 | `# {family_name} 家庭资产配置剧本` |
| `生成时间` | UTC 时间戳 |
| `方法论版本` | 如 `handbook-v0.1` |
| `风险偏好` | `conservative/balanced/aggressive` |
| 免责声明 | 明确说明不构成投资建议 |

### 示例

```markdown
# 王先生 家庭资产配置剧本

**生成时间:** 2026-06-28T12:00:00Z | **方法论版本:** handbook-v0.1 | **风险偏好:** balanced

> 本剧本不构成投资建议。所有数值基于方法论假设,实际配置需结合专业人士意见。
```

### 说明

历史文档里曾出现 `剧本版本`、`客户档案 ID`、`海外模块`、`当前阶段` 等更丰富元数据。当前引擎没有稳定输出这些字段,因此本章不再把它们定义为必需元数据。

---

## 4. 当前章节结构

当前剧本稳定包含以下主体结构:

1. `综合建议摘要`
2. `A. 客户情况概览`
3. `B. 资产推演`
4. `C. 资产配置执行方案`
5. `C3. 心理账户余额(P50)`
6. `C4. 心理账户余额(按阶段着色)`
7. `C5. 各层余额时序堆叠`
8. `C6. 各层余额与资金来源`
9. `剧本变更记录`

这是一份**以 C 段为核心展开的执行型剧本**,而不是旧版的 A-E 五段纯叙事模板。

---

## 5. 各段说明

### A. 客户情况概览

作用:

- 汇总家庭净资产
- 汇总当前收支状态
- 展示退休后收支口径
- 展示事件时间线

### 综合建议摘要

作用:

- 用自然语言总结当前家庭财务状态
- 给出重大节点应对、资产增值与现金流分配顺序
- 仅做“保障结构缺失分析”,不做具体保险产品推荐或目标保额测算

### B. 资产推演

作用:

- 以当前金融资产和年度净结余为起点
- 展示未来节点前后的累积资产变化
- 标出障碍型事件与阶段内现金流

这部分是“为什么当前家庭会面临哪些节点压力”的解释层。

当前图表口径补充:

- 含投资收益主图会展示到 `measurement_end_year`
- `measurement_end_year` 当前默认取问卷中最后一个重大节点年份;若无重大节点,回退到 `当前年份 + 30`

### C. 资产配置执行方案

作用:

- 把资金分成若干心理账户 / bucket
- 区分立即划拨与年度净结余分配
- 让顾问和客户知道“哪些钱先留,哪些钱后配”

典型内容:

- `C1. 初始存量资金分配(立即执行)`
- `C2. 年度净结余分配(按年预算,按月执行)`

当前默认顺序:

- 先应急
- 再最早到期节点
- 后续节点按时间顺序推进
- 最后富余资金

这部分是当前剧本最接近“顾问执行说明”的段落。

### C3 / C4 / C5 / C6

这些是 C 段的可视化与表格补充:

- `C3`:各心理账户年度 P50 余额
- `C4`:按阶段着色的心理账户余额表
- `C5`:整体时序堆叠图
- `C6`:各 bucket 的资金来源与扇带图

它们不是独立主题,而是服务于 C 段解释。

### 剧本变更记录

当前引擎会在末尾补一段变更记录,用于标识当前剧本是自动生成的哪一版。

---

## 6. 当前剧本的风格定位

当前剧本的主要风格是:

- **顾问可解释**
- **客户可阅读**
- **偏结构化**
- **偏执行讨论**

它不是:

- 具体产品推荐书
- 完全散文化的人生叙事
- 只给出权重区间、几乎不落地的概念稿

---

## 7. 与边界声明的关系

当前剧本必须保留以下边界:

- 不构成投资建议
- 不承诺收益
- 不替代律师、税务师、精算师等专业意见

具体边界详见 [`06-boundaries.md`](06-boundaries.md)。

---

## 8. 与旧版定义的差异

旧版文档曾将剧本定义为:

- 五段叙事
- 更强调“保守/平衡/进取”三骨架展示
- 更少的图表和 bucket 追踪

当前实现已经演化为:

- 结构化元数据 + A/B/C 主段
- 以资金分层和节点推演为核心
- 多个 C 段补充图表

因此,后续所有对剧本的说明应以本章为准,不再引用旧版 A-E 五段模板作为当前产品定义。
